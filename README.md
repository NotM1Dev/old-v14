## Usage

```sh
git clone https://github.com/m1-dev/v14
npm install
node .
```

## Features

- [Advanced command/event handler](https://commandkit.js.org)
- Useful utility commands e.g. steal emojis and kick command
- Full VC music system with /play, /pause, /queue etc.
- Leveling system (WIP)
